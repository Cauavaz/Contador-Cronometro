## 🚀 Desenvolvi um Contador/Cronômetro Interativo com React e 3D!

Estou animado para compartilhar meu mais recente projeto pessoal, o **CHRONOS**, uma aplicação web que explora a fusão de interfaces reativas com design 3D imersivo. O objetivo era ir além de um simples contador e criar uma experiência de usuário memorável.

### ✨ Destaques do Projeto:

- **Interface 3D com Spline**: Implementei um modelo 3D interativo que responde a eventos do usuário através do web component `<spline-viewer>`. A integração foi feita diretamente no React sem dependências adicionais, carregando a cena 3D de forma assíncrona a partir da URL pública do Spline. Isso permitiu criar uma experiência visual imersiva mantendo alta performance.

- **Animações Avançadas com GSAP e Motion**: Desenvolvi um sistema de animações sofisticado combinando GSAP (para o cursor interativo com efeitos de parallax e hover) com Motion (para animações de componentes React). O cursor personalizado reage dinamicamente à proximidade de elementos interativos, criando uma experiência mais fluida e moderna.

- **Arquitetura Robusta com React 19**: Construí uma arquitetura modular baseada em componentes reutilizáveis como `ContadorCronometro`, `Hero`, `TargetCursor` e `VariableProximity`. Utilizei hooks customizados para lógica compartilhada e implementei um sistema de estado que permite transições suaves entre diferentes modos da aplicação.

- **Styling com TailwindCSS + DaisyUI**: Criei um sistema de design responsivo e customizado usando TailwindCSS em conjunto com DaisyUI para componentes pré-construídos. Implementei breakpoints específicos para diferentes dispositivos (360px, 435px, 480px, 768px) garantindo uma experiência consistente em mobile, tablet e desktop.

- **Performance e Build Otimizado**: Configurei Vite como bundler para builds ultrarrápidos e hot module replacement. Implementei lazy loading de componentes e otimizei re-renders para garantir performance máxima, mesmo com múltiplas animações e o modelo 3D rodando simultaneamente.

Este projeto foi um mergulho profundo em como as tecnologias de frontend modernas podem ser orquestradas para criar experiências digitais ricas e envolventes. Foi um desafio empolgante combinar a lógica reativa do React com a estética visual do Spline e GSAP.
