## 🚀 Desenvolvi um Contador/Cronômetro Interativo com React e 3D!

Estou animado para compartilhar meu mais recente projeto pessoal, o **CHRONOS**, uma aplicação web que explora a fusão de interfaces reativas com design 3D imersivo. O objetivo era ir além de um simples contador e criar uma experiência de usuário memorável.

### ✨ Destaques do Projeto:

- **Interface 3D com Spline**: Integrei um modelo 3D interativo diretamente na aplicação React. A cena é carregada de forma assíncrona usando o web component `<spline-viewer>`, o que torna a adição de elementos 3D surpreendentemente simples e performática.

- **Animações Avançadas com GSAP e Motion**: Desenvolvi um cursor personalizado que reage a elementos da UI e animações de componentes fluidas, proporcionando um feedback visual dinâmico e moderno.

- **Arquitetura Robusta com React**: Utilizei as melhores práticas de componentização do React para construir uma base de código modular, reutilizável e de fácil manutenção. O projeto já está rodando em React 19!

- **Styling com TailwindCSS**: Criei um design responsivo e customizado de forma rápida e eficiente, mantendo o código CSS limpo e organizado.

Este projeto foi um mergulho profundo em como as tecnologias de frontend modernas podem ser orquestradas para criar experiências digitais ricas e envolventes. Foi um desafio empolgante combinar a lógica reativa do React com a estética visual do Spline e GSAP.
